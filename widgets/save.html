Save Widget

Site: Specification Save Specification Examples Usecases

The Save Widget is designed to allow users to embed a single line of code inside their site, blog or application and instantly benefit from services that support the Save intent.

The Save intent allows users to persist an object to any service that supports "save". The object could be a complex object such as a document, image, music file or even something simple like an link that would save the page for later reading.
